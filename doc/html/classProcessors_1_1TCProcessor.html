void Processors::TCProcessor::processRecv ( String  payload)

This function processes the input string which contains the OPCODE to concerned along with the action to take. This function will set the internal error variable accordingly, if there is an error of course. This function calls the process function which manages command execution, enabling, disabling, etc. The payload must be formatted as follows (in chars): xxxy where x is the OPCODE on 3 chars and y is the action on 1 char. The action is one of the following:

  • 1: execute
  • 3: enable
  • 5: disable

For example, to EXECUTE OPCODE 15, this function must receive 0151.

Parameters
payloadinput passed on by the GlobalReceiver in order to process the desired command.
