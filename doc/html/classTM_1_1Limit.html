Member Enumeration Documentation

enum TM::Limit::LIMIT_TYPE

defines if this limit is a HIGH limit or a LOW limit.

Enumerator:
NONE 

this is only used for initialization of the empty constructor.

HIGH 

this defines a high limit, such as red high or yellow high. It will check that the provided value is less than the limit.

LOW 

this defines a low limit, such as red low or yellow low. It will check that the provided value is greater than the limit.
